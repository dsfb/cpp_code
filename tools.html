network

  • POCO library - the framework; network, tcp ip, http, logs, utf8

  • winsock - windows sockets tcp ip library

linux system functions

  • unistd - to create a fork process, execute a command and get the system information
  • signals /usr/include/signal.h - to send signals (such like SIGTERM, SIGUSR1) from a user to an app and define how the app will react on them

to debug and analize binary files

  • cutter - the decompiler c++ and beta debugger based on radare2
  • strings - this is GCC tool to show strings inside a binary file
  • nm - to show function names in a binary file unless the binary file is stripped
  • gdb - gnu debugger for linux
  • x86dbg - open source debugger
  • Fidder - to analize http traffic
  • WireShark - to analize tcp/ip, sip and other traffic
  • ghidra - c++ decompiler
  • ida - the debugger and c++ decompiler
  • radare2 - UNIX-like reverse engineering framework and command-line toolset

testing

  • google tests - c++ library to perform tests
  • Fake Function Framework for C - fake function framework - to imitate functions

to create own web site from scratch

  • git pages + jekyll (the static site generator) https://jekyllrb.com/docs/

virtualization

  • VcXsrv - the app for Windows to show Linux GUI apps that was ran with WSL (add to .bashrc: export DISPLAY=localhost:0.0)
